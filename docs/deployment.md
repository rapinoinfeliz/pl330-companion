# Deploy Cloudflare

Crie D1, substitua o ID no `wrangler.jsonc`, aplique migrations e grave `ADMIN_SECRET` com Wrangler. Configure `APP_ORIGIN` para o endereço `workers.dev`, execute o build e `pnpm deploy`. Os quatro crons estão em UTC. Não ative plano pago, R2, domínio, Argo ou serviços de teste.
